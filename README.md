# OptLang
A stack based language all about learning code generation and optimization to me.

Name by [@slmnkara](https://github.com/slmnkara)

### Goals for this project

- [ ] Compiled to target triple of my machine (gas-aarch64-macos)
- [ ] Turing Complete
- [ ] Static typing
- [ ] Self hosted
- [ ] Optimized to a convenient degree for the developer
- [ ] Has inline assembly features akin to FreePascal
- [ ] All of the above with no added dependencies. Or in my case "Crate"s.
